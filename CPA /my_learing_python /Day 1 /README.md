### **1. 변수 선언 및 기본 자료형**

#### **관련 이론**

\*\*변수(Variable)\*\*는 값을 저장하는 메모리 공간에 붙이는 이름입니다. `변수명 = 값` 형태로 선언하고 할당합니다.

**기본 자료형**:

  * **정수형 (`int`)**: 양의 정수, 음의 정수, 0. (예: `10`, `-5`, `0`)
  * **실수형 (`float`)**: 소수점이 있는 숫자. (예: `3.14`, `-0.5`)
  * **문자열 (`str`)**: 문자들의 나열. 작은따옴표 `''`나 큰따옴표 `""`로 감쌉니다. (예: `'hello'`, `"Python"`)
  * **불린형 (`bool`)**: 참(`True`) 또는 거짓(`False`)만을 나타내는 논리값. 조건문에서 주로 사용됩니다.

#### **💡 암기 요소**

  * 기본적인 자료형의 이름과 형태
  * 변수 선언 문법 (`변수명 = 값`)

#### **연습 문제**

#### **문제 요구사항**
1.  `num1`이라는 변수에 `10`을, `num2`라는 변수에 `20`을 할당하세요.
2.  `result`라는 변수에 `num1`과 `num2`의 합을 할당하세요.
3.  `f"10 + 20 = 30"` 형식으로 결과를 출력하세요.

#### **가이드**
`f-string`을 사용하면 변수와 문자열을 결합하기 쉽습니다. `{}` 안에 변수 이름을 넣으면 해당 변수의 값이 출력됩니다.

### **2. 핵심 자료형 1 - 리스트 (List)**

#### **관련 이론**

**정의**: 여러 값을 순서대로 저장하는 \*\*변경 가능(mutable)\*\*한 컬렉션입니다. 대괄호 `[]`를 사용합니다.

**주요 특징**:

  * **순서(Ordered)**: 각 요소는 0부터 시작하는 인덱스를 가집니다.
  * **변경 가능(Mutable)**: 한 번 생성된 리스트의 요소를 추가, 삭제, 수정할 수 있습니다.
  * **다양한 타입**: 하나의 리스트에 정수, 문자열, 심지어 다른 리스트나 딕셔너리 등 서로 다른 타입의 요소를 저장할 수 있습니다.

#### **💡 암기 요소**

  * **생성**: `[]`, `list()`
  * **접근**: `리스트[인덱스]`, `리스트[시작:끝]` (슬라이싱)
  * **추가**: `리스트.append(요소)`, `리스트.insert(위치, 요소)`
  * **삭제**: `리스트.remove(값)`, `리스트.pop(인덱스)`, `del 리스트[인덱스]`
  * **길이**: `len(리스트)`
  * **정렬**: `리스트.sort()` (원본 변경), `sorted(리스트)` (새 리스트 반환)
  * **리스트 컴프리헨션**: `[표현식 for 항목 in 이터러블 if 조건]` (매우 중요\!)

#### **핵심 기록**

  * 리스트에서 요소를 삭제하면 그 뒤의 모든 요소 인덱스가 1씩 당겨집니다.

#### **연습 문제**

#### **문제 요구사항**
1.  `scores = [85, 92, 78, 90]` 리스트를 선언하세요.
2.  `scores` 리스트에 `65`를 **추가**하세요.
3.  `scores` 리스트에서 `92`라는 값을 **삭제**하세요.
4.  `scores` 리스트를 오름차순으로 **정렬**하세요.
5.  최종 리스트를 출력하세요.

#### **가이드**
리스트에 요소를 추가할 때는 `append()`, 값을 삭제할 때는 `remove()`, 리스트를 정렬할 때는 `sort()` 메서드를 사용해야 합니다.

### **3. 핵심 자료형 2 - 딕셔너리 (Dictionary)**

#### **관련 이론**

**정의**: \*\*키(key)\*\*와 \*\*값(value)\*\*의 쌍으로 데이터를 저장하는 컬렉션입니다. 중괄호 `{}`를 사용합니다.

**주요 특징**:

  * **키-값 쌍**: 각 값은 고유한 키로 식별됩니다. 키는 중복될 수 없으며, 변경 불가능한(immutable) 객체(문자열, 숫자, 튜플)여야 합니다.
  * **순서(Ordered)**: 파이썬 3.7+부터는 키-값 쌍의 삽입 순서가 유지됩니다. (이전 버전에서는 순서가 보장되지 않음)
  * **변경 가능(Mutable)**: 키-값 쌍을 추가, 삭제, 수정할 수 있습니다.

#### **💡 암기 요소**

  * **생성**: `{}`, `dict()`
  * **접근**: `딕셔너리[키]`, `딕셔너리.get(키, 기본값)` (키가 없을 때 오류 방지)
  * **추가/수정**: `딕셔너리[새_키] = 값`, `딕셔너리[기존_키] = 새_값`
  * **삭제**: `del 딕셔너리[키]`, `딕셔너리.pop(키)`
  * **키/값/아이템 얻기**: `딕셔너리.keys()`, `딕셔너리.values()`, `딕셔너리.items()`
  * **순회**: `for key in dict:`, `for value in dict.values():`, `for key, value in dict.items():`

#### **핵심 기록**

  * 리스트의 `pop()`과 달리 딕셔너리의 `pop()`은 키가 없을 때 `default_value`를 지정하여 오류를 방지할 수 있습니다. `del`은 이런 기능이 없습니다.
  * `get()`, `pop()`, `keys()` 같은 기능들은 모두 딕셔너리에 속한 **메서드**이므로 `.` (점)을 사용해서 호출합니다.

#### **연습 문제**

#### **문제 요구사항**
1.  `product = {'name': '컴퓨터', 'price': 1200000}` 딕셔너리를 선언하세요.
2.  `product` 딕셔너리에 `'재고': 100` 키-값 쌍을 **추가**하세요.
3.  `product` 딕셔너리의 `'price'` 값을 `1150000`으로 **수정**하세요.
4.  `for` 루프와 `.items()` 메서드를 사용해 딕셔너리의 모든 키와 값을 `"키: 값"` 형식으로 출력하세요.

#### **가이드**
* 딕셔너리에 새로운 키-값 쌍을 추가하거나 기존 값을 수정할 때는 `딕셔너리[키] = 값` 문법을 사용합니다.
* 딕셔너리를 `for` 루프로 순회할 때는 `for key, value in 딕셔너리.items():` 문법을 사용하면 키와 값을 동시에 얻을 수 있습니다.

### **4. 튜플(Tuple)과 집합(Set) (기본 개념)**

#### **관련 이론**

  * **튜플 (Tuple)**: 리스트와 유사하지만 \*\*변경 불가능(immutable)\*\*한 순서 있는 컬렉션입니다. 소괄호 `()`를 사용합니다.
      * 주로 사용되는 경우: 함수의 반환 값, 데이터의 불변성을 보장해야 할 때.
  * **집합 (Set)**: 중복을 허용하지 않고 순서가 없는 컬렉션입니다. 중괄호 `{}`를 사용합니다 (단, 빈 집합은 `set()`으로 생성).
      * 주로 사용되는 경우: 중복 제거, 교집합/합집합 등 집합 연산.

#### **💡 암기 요소**

  * **튜플**: `()` 또는 쉼표로 구분 (예: `(1, 2, 3)`, `1, 2, 3`), **변경 불가능**.
  * **집합**: `{}` (빈 집합 `set()`), **중복 없음**, 순서 없음.

#### **연습 문제**

#### **문제 요구사항**
1.  `data = (100, 200, 300)` 튜플을 선언하세요.
2.  이 튜플을 `a, b, c` 세 개의 변수에 **언패킹(Unpacking)**하여 할당하세요.
3.  `numbers = [1, 2, 3, 2, 1]` 리스트를 선언하세요.
4.  `numbers` 리스트의 **중복을 제거**한 결과를 출력하세요.

#### **가이드**
* 튜플 언패킹은 변수의 개수와 튜플 요소의 개수가 일치해야 합니다.
* 리스트의 중복을 제거하려면 `set()` 함수를 사용하여 리스트를 집합으로 변환하는 방법을 활용할 수 있습니다.

### **5. 제어문 - `for` & `while` 반복문**

#### **관련 이론**

  * **`for` 루프**: 리스트와 같은 컬렉션의 모든 요소를 순회하며 코드를 실행합니다.
  * **`while` 루프**: 특정 조건이 `True`인 동안 반복하며, 무한 루프에 빠지지 않도록 루프 내에서 조건을 변경해야 합니다.

#### **💡 암기 요소**

  * `for` 루프와 `range()` 함수 사용법
  * `while` 루프의 조건 변경 필수
  * `break`와 `continue`의 역할

#### **연습 문제**


#### **문제 요구사항 (두 문제)**
1.  `numbers = [1, 2, 3, 4, 5]` 리스트를 선언하세요. **`for` 루프**를 사용해 이 리스트를 순회하며, **짝수일 때만** 해당 숫자를 출력하세요.
2.  사용자로부터 숫자를 계속 입력받아 합계를 구하다가 **`0`을 입력하면 루프를 종료**하고, 최종 합계를 출력하는 **`while` 루프**를 작성하세요.

#### **가이드**
* **`for` 루프:** `if` 조건문과 함께 사용하여 특정 조건을 만족하는 요소만 처리할 수 있습니다.
* **`while` 루프:** 사용자 입력을 받을 때는 `input()` 함수를 사용하고, 입력받은 문자열을 숫자로 변환하려면 `int()` 함수를 사용해야 합니다. 루프를 종료할 조건(`0` 입력)과 무한 루프에 빠지지 않도록 변수를 잘 제어하는 것이 중요합니다.
