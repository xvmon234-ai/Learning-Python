# 삼일회계법인 신입공인회계사 디지털 전형 - 1일차 모의 면접

본 문서는 삼일회계법인 신입공인회계사 디지털 전형의 1일차 모의 면접 과제들을 요약한 것입니다. 각 과제는 주어진 시간 내에 문제 풀이 및 질의응답을 통해 지원자의 디지털 역량과 회계 전문성을 평가합니다.

---

## 공통 안내 사항

* **준비 시간:** 각 과제당 5분 (문제 분석)
* **풀이 시간:** 각 과제당 20분 (AI 활용 가능, GPT 적극 활용 권장)
* **질의 시간:** 각 과제당 10~15분 (수행 과제에 대한 면접관 질의)

---

## 과제 목록

### 1. 1일차 - 1회차: 데이터 정제 및 이상치 탐지

* **주제:** 회계 데이터 정제 및 이상치 탐지
* **개요:** 복잡한 회계 거래 데이터에서 오류와 이상 징후를 식별하고 정제하는 스크립트 개발. 특히, 중복 거래, 필수 정보 누락, 비정상적인 금액 등을 탐지하고 처리.
* **주요 요구사항:** 데이터 불러오기, 결측치/중복값 처리, 금액 이상치(IQR 기준) 탐지, 특정 기간/거래 유형별 집계 분석.

### 2. 1일차 - 2회차: 데이터 분석 및 시각화

* **주제:** 재무 데이터 분석 및 시각화
* **개요:** 제조 기업의 월별 판매 데이터를 분석하여 매출 추이와 제품별 기여도를 파악하고, 주요 재무 의사결정을 위한 시각화 자료 작성.
* **주요 요구사항:** 데이터 전처리(날짜, 매출액 계산), 월별 총 매출액 추이 분석 및 꺾은선 그래프 시각화, 제품 카테고리별 매출 기여도 분석 및 파이 차트 시각화, 지역별 상위 3개 제품 카테고리 분석.

### 3. 1일차 - 3회차: 재무 모델링 관련 코딩 (NPV 및 IRR 계산)

* **주제:** 재무 모델링 (투자 프로젝트 경제성 평가)
* **개요:** 특정 투자 프로젝트의 경제성을 평가하기 위한 간단한 현금흐름 할인(DCF) 모델 구축. 초기 투자액과 예상 현금흐름 데이터를 바탕으로 NPV(순현재가치)와 IRR(내부수익률) 계산.
* **주요 요구사항:** 초기 데이터 정의, NPV 계산 함수 작성, IRR 계산 (라이브러리 활용), 계산 결과 출력 및 경제적 타당성 해석.

### 4. 1일차 - 4회차: 재무 데이터 분석 및 회계 감사 자동화 (가상 시나리오)

* **주제:** 재무 데이터 분석 및 회계 감사 자동화
* **개요:** 고객사의 매출채권 데이터를 분석하여 이상 징후를 식별하고, 감사 절차를 자동화하는 스크립트 개발. 회수 기간이 길거나, 특정 고객사 채권 잔액이 급격히 변동하는 경우 식별에 중점.
* **주요 요구사항:** 데이터 전처리(날짜 변환), 매출채권 회수 기간 분석(평균 대비 30일 초과), 장기 미회수 매출채권 식별(60일 이상 경과), 고객별 채권 잔액 변동 분석 및 채권 집중 고객 식별.
